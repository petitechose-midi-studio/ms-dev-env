# Ideas - Idées à développer

Ce dossier contient les idées, réflexions et propositions à explorer pour le projet.

## Structure

| Fichier | Description |
|---------|-------------|
| `README.md` | Ce fichier |
| `backlog.md` | Liste des idées en vrac, à trier |

## Processus

1. **Capture** - Ajouter l'idée dans `backlog.md`
2. **Développement** - Si l'idée mérite exploration, créer un fichier dédié
3. **Promotion** - Si l'idée est validée, la déplacer vers le dossier approprié :
   - `architecture/` pour les décisions d'architecture
   - `feature-xxx/` pour les features planifiées
